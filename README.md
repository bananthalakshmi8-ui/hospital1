# May i help you 

A complete frontend-only Hospital + Medical eCommerce website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS 3
- Framer Motion
- React Icons
- React Router DOM

## Features

- 12 fully designed pages
- Responsive mobile-first design
- Dark mode toggle
- Shopping cart & wishlist (localStorage)
- Toast notifications
- Glassmorphism UI
- Framer Motion animations
- FAQ, testimonials, animated counters

## Pages

1. Home
2. Pharmacy Store
3. Product Details
4. Doctors
5. Appointment Booking
6. Lab Tests
7. Cart
8. Checkout
9. User Dashboard
10. About
11. Contact
12. Prescription Upload

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
